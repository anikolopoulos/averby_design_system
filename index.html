Table Guidelines

Best Practices

  • • Keep headers concise: Use clear, short column labels
  • • Align data types: Numbers right, text left
  • • Use consistent formatting: Dates, currency, numbers
  • • Provide sorting: For columns with sortable data
  • • Show loading states: Skeleton loaders for better UX

Responsive Strategy

  • • Mobile cards: Transform to card layout < 768px
  • • Hide columns: Show only essential data on small screens
  • • Horizontal scroll: For data-heavy tables
  • • Pagination: Limit rows for performance
  • • Touch targets: Ensure 44px minimum for mobile actions
